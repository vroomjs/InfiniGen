Free AI Service Playground

Use these endpoints to add AI to your site without API keys.

GET /v1/chat

fetch("http://localhost:3000/v1/chat?prompt=Hello")

POST /v1/chat

fetch("http://localhost:3000/v1/chat", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    messages: [{ role: "user", content: "Hello" }],
    stream: true
  })
});

Streaming Example

const evtSource = new EventSource("http://localhost:3000/v1/chat?prompt=Hello&stream=true");
evtSource.onmessage = (e) => {
  if(e.data === "done") evtSource.close();
  else console.log(e.data);
};
